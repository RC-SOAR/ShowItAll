## Description
ShowItAll displays all the basics in a single widget. So you spend less time widgeting and more time flying.

- Compatible with most colour screen radios
- Essential info condensed in a single widget
- Autodetects voltage telemetry (Cels, RxBt, A1, A2, A3 or A4)
- Displays link quality telemetry for ELRS and UNI receivers
- Optional 'motor armed' alert for electric models
- Choice of theme or custom colours

## Requirements
- Transmitter with colour screen with width 480 or 800 pixels (TX16S Mk1/2/3, TX15, X10, X12 etc.)
- OpenTX 2.2+ or Edge 2.4+

## Installation

The latest release can be downloaded from:
https://github.com/RC-SOAR/ShowItAll/releases/latest
Download the .zip file as directed, and then follow the instructions in the included PDF.
